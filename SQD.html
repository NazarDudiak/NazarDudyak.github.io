  • Квадратне Рівняння

    • Квадратне рівняння — це рівняння вигляду ax2 + bx + c = 0, де a не дорівнює 0. Загальним розв'язком цього рівняння є формула X1,2=-b+-√D/2a

  • Дискримінант квадратного рівняння

    • Дискримінант D квадратного рівняння ax2 + bx + c = 0 рівний D = b2 − 4ac.
    • Якщо D = 0 єдиний корінь (точніше — два однакові корені)
    • Якщо D > 0 у цьому випадку дане рівняння має два корені.
    • Якщо D < 0 то у цьому випадку дане рівняння не має коренів взагалі.
